¿qué podemos ofrecerle?

1- Diseño de páginas web a medida
Realizamos su proyecto web desde cero. Según sus necesidades y gustos , adaptándonos a su imagen corporativa y utilizando nuestro sistema, con lo que no necesitará de terceros para actualizaciones o cambios en la página web.

2- Adaptación de plantillas:
Podemos realizar todo tipo de combinaciones entre las páginas web del catálogo. Cambiar un diseño por otro, secciones, funciones, cambiar colores,...
2.1- Combinación de modelos
2.2- Cambiar el color
2.3- Añadir o eliminar secciones o elementos
2.4- añadir logotipos
2.5- Ampliar el tamaño de las fotografías
2.6- Acceso restringido sólo para usuarios

3- Incluir música
Disponemos de un reproductor de música para canciones en mp3 sin límite de canciones y autogestionable por el propio usuario, que podemos insertar en cualquier modelo de nuestro catálogo de páginas web.

4- Incluir una galería de vídeos
Además de las galerías de vídeos que puede adquirir directamente en nuestro catálogo, también podemos incluir un preproductor de vídeos .flv o una galería de vídeos en cualquiera de los modelos de páginas web de nuestro catálogo.
